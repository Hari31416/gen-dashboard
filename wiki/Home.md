# Home

AI Dashboard is designed to democratize data visualization by allowing anyone to generate complex, interactive dashboards using simple language.

## High-Level Vision
- **Natural Language First**: No SQL or chart configuration knowledge required.
- **Agentic reasoning**: Multi-stage AI pipeline for high-fidelity results.
- **Interactivity**: Built-in drill-down and refinement capabilities.
- **Security**: Strict SQL sanitization and Pydantic validation.
